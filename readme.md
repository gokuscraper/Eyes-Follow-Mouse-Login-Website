# 登录界面眼睛跟随鼠标的网站

一个超轻量、纯前端、带交互动画的登录页面。

其最大的亮点是页面左侧的动漫小人，它们的眼睛会**实时跟随你的鼠标移动**，

当你输入密码时，它们还会做出相应的表情和动作，极具灵动感和交互性。

![image-20260317024541761](https://asiaassets.gokuscraper.com/images/2026/03/17/c751c2af25dc3512.webp)

这个项目是从一个复杂的全栈 Next.js 应用中**剥离**出来的。

去掉了所有后端依赖、数据库连接和 AI 逻辑，只保留了最精华的 UI 部分，

并转换成了一个**单文件的 `index.html`**。
